# SOCIAM barebones stream consumer

Barebones multi-stream consumer for wikipedia/twitter/other things.

## Server

* Node.js
* Socket.IO

To run:

 cd server
 npm install
 node app

## Client

* Angular.js
* D3
* Cubism
* Socket.IO

To run: 

 cd client;
 http-server
 and open a browser to localhost:8080
